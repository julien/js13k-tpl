js13k-tpl
---------

Template project for #js13k games.

Requirements
------------

- [nodejs](http://nodejs.org)

*Tested with `v8.2.1`*

Install dependencies
--------------------

```shell
npm i
```

Available scripts
-----------------

- start a local http server with

```shell
npm start
```

- build everything with

```shell
npm run build
```
